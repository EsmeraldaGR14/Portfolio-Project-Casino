Casino

Single Player
Rules

  1. Higher number of cards: 3 points

  2. Higher number of spades: 1 point

  3. Diamond number 10: 2 point

  4. Spades number 2: 1 point

  5. Each clearing of the board known as a "sweep": 1 extra point
